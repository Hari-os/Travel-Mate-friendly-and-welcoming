# Travel Website Landing Page

A simple and responsive travel website landing page created using **HTML** and **CSS**. This project is designed to showcase travel destinations, services, and contact information in a clean and visually appealing way.

## Features
- Fully responsive design
- Attractive header navigation menu
- About section with university-related content
- Services section showcasing offerings
- Gallery with beautiful travel images
- Contact details and footer section

## Technologies Used
- **HTML5** for structure
- **CSS3** for styling
- Images and icons for UI enhancement
